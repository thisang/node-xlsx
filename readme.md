# 使用「node-xlsx」格式化 xlsx 文件 #

功能:

- 批量格式化 `./input`目录下的 `.xlsl`文件

- 格式:

  | code |  a   | pageId |  q1  | q2~qn |
  | :--: | :--: | :----: | :--: | :---: |
  |  1   | ...  |  123   | ...  |  ...  |
  |  2   | ...  |  124   | ...  |  ...  |

  ​

输出:

- `./output`

- 格式:

  |  q   |  a   |
  | :--: | :--: |
  | ...  | ...  |
